## 📌 Project Overview
Customer behaviour and revenue optimization analysis on **500K+ e-commerce transactions** using Excel and Power BI. Includes data cleaning, EDA, RFM segmentation and interactive dashboard.

---

## ❓ Business Questions Answered
- Which customers generate the most revenue?
- Which countries drive the highest sales?
- What time of year generates maximum revenue?
- Which products sell the most?
- Which customers should be targeted for retention?

---

## 🔍 Key Findings
| Metric | Value |
|--------|-------|
| Total Revenue | £8.76 Million |
| Total Transactions | 396,370 |
| Total Customers | 4,335 |
| VIP Customer Revenue Share | 87.2% |
| Peak Revenue Months | Sept, Oct, Nov |
| Top Market | United Kingdom (83%) |
