# E-Commerce Customer Intelligence Case Study

## Project Overview
Customer behaviour and revenue optimization analysis 
on 500K+ e-commerce transactions using Excel and Power BI.

## Business Questions Answered
- Which customers generate the most revenue?
- Which countries drive the highest sales?
- What time of year generates maximum revenue?
- Which products sell the most?
- Which customers should be targeted for retention?

## Tools Used
- Microsoft Excel — Data cleaning and analysis
- Power BI — Interactive dashboard

## Key Findings
- Total Revenue: £8.76 million
- VIP customers (29%) generate 87% of total revenue
- Revenue peaks in September, October, November
- UK dominates with 83% of total revenue
- Top products are home decor and gifting items
